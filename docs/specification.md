HomeHelp Connect - Product Specification (MVP)
1. Product Overview
Product Name
HomeHelp Connect
Tagline
Find Help. Find Work.
Purpose
HomeHelp Connect is a platform that connects residents looking for household help with trusted individual workers such as maids, cooks, baby-care helpers, and elder-care helpers.
The initial launch is community-focused but the platform is designed to scale to cities and regions in the future.
________________________________________
2. Goals
Resident Goals
•	Find trusted workers
•	Search workers by service and availability
•	Contact workers directly
•	Post requirements when suitable workers are unavailable
•	Review workers after hiring
Worker Goals
•	Create a profile
•	Showcase services and charges
•	Display availability
•	Receive work opportunities
•	View resident interest requests
________________________________________

•	Login using Google
•	Search workers
•	View worker profiles
•	Call workers
•	Contact workers via WhatsApp
•	Mark interest in workers
•	Save workers
•	Post requirements

Worker
Can:
•	Login using Mobile OTP
•	Create and manage profile
•	Set availability
•	Set service charges
•	Toggle “Looking For Work”

•	View resident interest requests
________________________________________
Admin
Can:
•	View platform statistics
•	Manage users
•	Disable fake profiles
•	Moderate reviews
•	Manage worker categories and services

4. Location Model

Country → State → City → Area → Community (Optional)

India Tamil Nadu Chennai Adyar IITM

•	Push Notifications

22. MVP Success Criteria
The MVP is successful if:
1.	Workers can create profiles.
2.	Residents can find workers.
3.	Residents can contact workers.
4.	Residents can post requirements.
5.	Workers can find opportunities.
6.	Reviews can be collected.
7.	At least one successful worker-resident connection occurs. ““”
