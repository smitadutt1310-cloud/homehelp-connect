# HomeHelp Connect - Testing Strategy

## MVP Testing

### Manual Testing

Resident Flow

* Login
* Search Worker
* View Profile
* Contact Worker

Worker Flow

* Login
* Create Profile
* Update Availability
* View Requirements

Admin Flow

* Login
* View Dashboard

## Device Testing

* Android Chrome
* iPhone Safari
* Desktop Chrome

## Success Criteria

A resident can find and contact a worker successfully.

A worker can create a profile successfully.

A resident can post a requirement successfully.
